# 2019-HACKATON-RSV43
Программный код для регионального хакатона

Технологии:
~~~~
Python - транслятор языка программирования с открытым исходным кодом
Flask - фреймворк для разработки WSGI-приложений
Gunicorn - WSGI-сервер для распределения входящих запросов между воркерами
Nginx+SSL - в роли проксирующего веб-сервера и для отдачи статики
supervisord - сервис перезапуска процессов
ZMQ - передача сигналов между процессами
Redis - передача и кеширование данных, не критичных к перезагрузке системы
PostgreSQL - сохранение данных с поддержкой шардинга и реплицирования
~~~~
